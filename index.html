The definitive badge list for all of your projects.

A curated list of Shield.io badges, organized in lists. Simply copy the code to use it in your own project! ⬇️

What You Should Know:

Open Source

This project is maintained and new badges are constantly being added. Feel free to submit a pull request if you want to add your own badge!

Customizable

Customizable badges offer versatile visual elements, allowing you to modify content, colors, logos, and styles to suit your needs. See the navigation menu above!

Thanks to the team behind shield.io, badges have become an integral part of modern software development, especially in open-source projects and repositories. Over the last few years, the use of badges has significantly increased due to their ability to provide quick, visual information about various aspects of a project's status, health, and functionalities.

How to Use:

In order to use this list of badges correctly, navigate to the badge you want in the md-badges repository, and copy the "Markdown Code" text.

FAQ

Frequently Asked Questions

How do I use this?

Go to the GitHub repository by clicking "View On GitHub".

Look for the badge you would like to add and copy the text in the "Markdown Code" section.

Tip: Use 'Ctrl + F' or the Table Of Contents to search quickly for your badge.

Why won't my badge lead anywhere?

By default, (mostly) all badges have (#) at the end of their code to define a link.

You need to replace the (#) with a link that you'd like the badge to go to when clicked on.

For example, if the code is:

[![GitHub](https://img.shields.io/badge/GitHub-%23121011.svg?logo=github&logoColor=white)](#)

...you'll need to replace the (#) with a link (eg. https://github.com/inttter).

If the code is a dynamically updating badge such as this one:

 [![GitHub release](https://img.shields.io/github/release/badges/shields.svg)](https://github.com/badges/shields/releases)

You must replace 'badges' with your GitHub username/organization, and 'shields' with your repository name. This includes the redirect URL aswell.

Can I customize these badges?

Yes. Please refer to the Styles and Customization documentations for how to style and customize your badges.

Can I use badges on any repository hosting platform?

Yes, as long as they support Markdown rendering.

If you need more info, consult the help page for your platform.
